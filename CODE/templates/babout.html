IMAGE BASED STRESS DETECTION USING DEEP LEARNING

About this project

Data set gathering

The dataset for the Face expressions images is collected from the GitHub website.

The size of overall dataset is 805 MB. From which we have are splitting the data into (Training_images, Testing_images), where the Training_images dataset size is 584 MB and Testing_images size is 221 MB.

Training Data

We are using the pre-processed training dataset to train our model using CNN algorithm. CNN algorithm consists of 4 layers: Input layer, Convolution Layer, pooling layer, Flatten layer and dense layer.

In input layer we consider images as input.

In Convolution layer, we convert image into matric format. Here matrix size is 1024 X 1024 (rows X columns).

In the pooling layer the numerical values will be stored. To change the numerical data to binary data, we use machine learning algorithm named Softmax (supervised learning algorithm). In Softmax layer we will convert the numerical data to binary.

In flatten layer and dense the classes of total dataset (7 types) is stored which will be in the binary data format.

We use fit_generator method for saving the data in the form of .h5. Here model is a format for storing the binary data.

Testing

The image is uploaded which is to be tested. After uploading an image DWT is applied from which we get coefficients (approximation, horizontal, vertical & diagonal) of the image. Then we will give the image to saved model and predict the output.
